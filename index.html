Why Appitat?

Striking Design

With an all-metal brushed enclosure, ambiently illuminated top, and LED accents, Appitat is a computer you don't have to hide.

Appitat is user serviceable, too. Remove the bottom cover to access the SD Card, Bluetooth / Wi Fi module, and other system components.

Well Connected

Appitat seamlessly connects to your home internet service, making apps available for you to access from anywhere in the world. Simply plug it in, turn it on, and you'll be up and running in seconds.

When you install an app, you can explicitly allow or deny it access to other apps / services on the Internet. And if you decide to completely remove an app, its data is securely erased forever. With Appitat, you're in complete control of your data.

Developer Friendly

Stop worrying about infrastructure. With Appitat, scalability is a non-issue. Simply wrap your web app inside a Docker container, push it to the app market and it's available worldwide within seconds.

For paid apps, give us your Bitcoin address and we'll broker the transaction for you for a 1% fee. Forget about app approvals, payouts, and country-specific currencies.

Download the developer tools to get started.

Technical Details

Hardware

  • Power-efficient ARM-based System-on-a-Chip
  • 2 GB RAM
  • 8 GB storage via micro SD card
  • 10/100/1000 Ethernet
  • USB 3.0 port for expandable storage
  • 802.11 a/b/g/n Wi-Fi
  • Bluetooth 4.0

Software

  • Linux 3.14
  • Docker 0.10
  • SQLite 3 for persistence
  • Support for Golang 1.2, Node 0.10

Project Status

Appitat is currently in the prototype phase. We’re 3d printing models of the enclosure, finalizing hardware specifications, and nailing down the software / network architecture for the system. We expect to have Appitat ready by the end of 2014. In the meantime, enter your email address below to receive project updates. We’ll notify you when updates are available.
